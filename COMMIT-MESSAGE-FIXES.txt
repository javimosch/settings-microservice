### 1. Fixed CSP/Helmet blocking Alpine.js/Vue
- Removed helmet from server.js to prevent CSP errors

### 2. Added bulk retrieval routes for settings
New external API endpoints:
- GET /api/client-settings/all/:clientId - Get all settings for a client
- GET /api/user-settings/all/:userId - Get all settings for a user
- GET /api/dynamic-settings/all/:uniqueId - Get all settings for a uniqueId

### 3. Updated API controller
Added three new methods:
- getAllClientSettings() - Fetch all settings for specific clientId
- getAllUserSettings() - Fetch all settings for specific userId
- getAllDynamicSettings() - Fetch all settings for specific uniqueId
